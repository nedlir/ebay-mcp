# eBay API MCP Server

For full documentation, please visit the project repository:
https://github.com/YosefHayim/ebay-api-mcp-server

## Quick Start

1. Get eBay Developer credentials from https://developer.ebay.com/my/keys
2. Configure the server with your Client ID, Client Secret, and RU Name
3. Choose your environment (sandbox for testing, production for live)
4. Optional: Set up OAuth for higher rate limits (10k-50k requests/day vs 1k)

## Features

- 230+ eBay API tools covering all major Sell APIs
- Inventory management and bulk operations
- Order fulfillment and shipping
- Marketing campaigns and analytics
- OAuth 2.0 with automatic token refresh
- Type-safe with comprehensive validation
- Well-tested (870+ tests, 99%+ coverage)
